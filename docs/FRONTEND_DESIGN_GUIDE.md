# Frontend Design Guide — Inventra

> **Read this before any UI work.** This guide ensures every component is distinctive, polished, textured, and avoids generic AI aesthetics.

## Design Thinking

Before coding any UI, understand the context and commit to a BOLD aesthetic direction:
- **Purpose**: What problem does this interface solve? Who uses it?
- **Tone**: Inventra's tone is **Industrial Warmth** — playful but professional, grungy but tidy, dynamic but functional. Think hex.tech energy: creative humans built this, not a template generator.
- **Constraints**: Technical requirements (framework, performance, accessibility).
- **Differentiation**: What makes this UNFORGETTABLE? What's the one thing someone will remember?

**CRITICAL**: Execute with precision and personality. Every surface should have texture. Every animation should feel handcrafted. Every layout should feel opinionated. If it could have been generated by any AI tool, start over.

Then implement working code that is:
- Production-grade and functional
- Visually striking and memorable
- Textured and tactile — never flat or sterile
- Cohesive with Inventra's "Industrial Warmth" aesthetic
- Meticulously refined in every detail

## Frontend Aesthetics Guidelines

### Typography
Choose fonts that are beautiful, unique, and have *character*. Avoid generic fonts like Arial, Inter, and Roboto. For Inventra specifically: mix serifs and sans-serifs confidently. Use Fraunces (italic for hero headlines) + Space Grotesk (bold geometric sans-serif) as the primary display pairing — like hex.tech's hero where italic serif meets bold sans. Satoshi for warm body text. JetBrains Mono for data. Go BIG with headlines — dramatic scale differences between heading and body create visual impact.

### Color & Theme
**Warm cream + warm charcoal + dusty rose.** This is the hex.tech-inspired palette:
- Light mode canvas: warm cream (`#F5F1EC`), never pure white
- Dark mode canvas: warm charcoal with rose undertone (`#141113`), never cold blue-black
- Accent: dusty rose (`#C4918A`) — used sparingly for CTAs, labels, links, highlights
- Text: warm near-black (`#1C1917`) in light mode, warm cream (`#F5F0EB`) in dark mode
- **Never use pure `#FFFFFF` or `#000000`** — always warm-shifted
- Use dark section bands inside light pages for dramatic rhythm (like hex.tech's alternating light/dark sections)

See VISION.md for the complete color system with all CSS variables.

### Texture & Surfaces
**This is what sets Inventra apart.** Every background should feel physical:
- Apply subtle grain/noise overlays to page backgrounds (0.03-0.05 opacity)
- Use dot-grid patterns on hero sections (like hex.tech's subtle grid)
- Use warm cream tones, never sterile white
- Cards should have warm borders and soft shadows that feel tangible
- Alternate between warm background tones for section rhythm
- Dark sections can contrast against light ones for dramatic variety
- Floating elements use warm glass (backdrop-blur with warm-tinted semi-transparent backgrounds)

### Motion
Animations should feel handcrafted and surprising, never generic or formulaic. Key patterns:
- **Marquee tickers**: Continuous horizontal scroll on dark bars with dusty rose text
- **Staggered text reveals**: Headlines that animate word-by-word or letter-by-letter
- **Scroll-triggered entrances**: Elements choreographed as they enter viewport
- **Parallax layers**: Different scroll speeds for depth
- **Interactive demos**: Real UI that users can interact with, not static screenshots

Use Framer Motion for React component animations. Use GSAP + ScrollTrigger for scroll-driven animations and complex timelines. Focus on high-impact moments: one well-orchestrated section entrance creates more delight than scattered micro-interactions.

### Spatial Composition
Unexpected layouts. Asymmetry. Overlap. Grid-breaking elements. Generous negative space OR controlled density. Use industrial details: fine divider lines, pill-shaped badges in dusty rose, subtle borders that show precision and care. Uppercase tracked-out labels for section markers.

### Backgrounds & Visual Details
Create atmosphere and depth. **Never default to solid colors.** Apply:
- Grain/noise texture overlays (SVG or CSS-based, lightweight)
- Dot-grid patterns for hero/featured sections
- Warm-toned backgrounds that alternate between sections (cream ↔ deeper cream ↔ dark charcoal bands)
- Layered elements with shadows that suggest physical depth
- Decorative borders and divider lines with purpose
- Dark-on-light contrast sections for dramatic rhythm
- Marquee ticker bands as section dividers

## What to NEVER Do

Never use generic AI-generated aesthetics:
- Overused font families (Inter, Roboto, Arial, system fonts)
- Cliché color schemes (purple gradients, cold blue primaries, bright teal accents)
- Pure white (`#FFFFFF`) backgrounds with no texture or warmth
- Pure black (`#000000`) text or backgrounds — always warm-shift
- Cold blue-tinted dark modes — dark mode must have warm undertones
- Predictable layouts and component patterns
- Cookie-cutter design that lacks context-specific character
- Sterile, clinical, flat surfaces with no depth or grain
- Generic gradient blobs without purpose or context
- Animations that feel algorithmic or template-driven
- Hot pink, bubblegum pink, or neon accents — the dusty rose is muted and sophisticated

If it looks like every other SaaS landing page, it's wrong. If it looks like hex.tech's team could have built it with care, it's right.

## Matching Complexity to Vision

Match implementation complexity to the aesthetic vision. Inventra's "Industrial Warmth" direction means:
- Surfaces always have texture (grain overlays, dot-grid patterns, warm tones)
- Colors are always warm (cream, charcoal, dusty rose — never cold)
- Typography is bold and mixed (italic serifs with bold sans-serifs, dramatic scale)
- Animations are choreographed and feel human (staggered reveals, marquee tickers)
- Industrial details add precision (fine borders, pill badges, divider lines, uppercase labels)
- Dark section bands inside light pages create visual rhythm

## Inventra-Specific Direction

Refer to **VISION.md** for the complete design system, color palette, typography pairings, and motion principles.

**Primary Inspiration: Hex (hex.tech)** — The warm cream + charcoal + dusty rose color scheme. Dot-grid textures. Scrolling marquee tickers on dark bars. Real product UI as hero content. Bold italic-serif + geometric-sans type mixing. Animations that feel handcrafted and human.

**Secondary Inspiration: Notion** — Clean layouts, elegant simplicity, refined interactions, approachable professionalism.

**Core Aesthetic: "Industrial Warmth"** — Bold, confident, cutting-edge but approachable. Warm cream canvas with grain texture. Warm charcoal for depth and dark sections. Dusty rose accents for personality and interactivity. Geometric precision meets organic, playful motion.

Every surface should feel *physical*. Every animation should feel *intentional*. Every layout should feel *opinionated*. Every detail should feel *handcrafted*.

Don't hold back. Show what can truly be created when thinking outside the box and committing fully to a distinctive vision that real creative humans would be proud of.
